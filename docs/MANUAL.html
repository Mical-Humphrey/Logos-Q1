This guide explains how to install, configure, run, and interpret Logos-Q1 ‚Äî both as an education lab and a professional backtesting system.

1) Installation & Setup

Requirements: Python ‚â•3.11, pip, Git, modern browser.

git clone https://github.com/Mical-Humphrey/Logos-Q1
cd Logos-Q1
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Run tests: pytest ‚Äî ensures math & finance modules pass baseline checks.

2) Launching the Web Docs

Simply open docs/index.html in any browser to view your documentation suite. Each module (Math, Finance, Manual, System Design) is self-contained and styled consistently.

3) Running Backtests

Example command:

python -m logos.backtest --symbol TSLA --strategy momentum --start 2020-01-01 --end 2024-12-31
  • Results stored in runs/ directory.
  • Equity curve, metrics (CAGR, Sharpe, MDD) auto-generated as JSON + chart HTML.

4) Exploring Results

  • Equity curve: visualizes growth of simulated portfolio.
  • Metrics: summarized in runs/summary.json.
  • Trade log: CSV with timestamps, prices, position sizes, PnL.
Pro tip

Pair Logos-Q1 backtests with docs/MATH.html to understand why each signal behaves differently in equities, crypto, or FX context.

5) Education Mode

Education mode transforms Logos-Q1 into a self-teaching tool. It explains every step as it runs:

python -m logos.tutor --lesson mean_reversion

Outputs annotated charts, variable definitions, and example trades.

6) Troubleshooting

  • RuntimeWarning: NaN in subtract ‚Äî often due to incomplete data window. Solution: ensure enough bars before indicators initialize.
  • Empty backtest: verify --start / --end within dataset range.
  • Import errors: re-activate venv or reinstall requirements.
